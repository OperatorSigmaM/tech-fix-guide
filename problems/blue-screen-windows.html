What is Blue Screen of Death (BSOD)?

The Blue Screen of Death (BSOD) is Windows' critical error screen that appears when the operating system encounters a fatal error it cannot recover from. The system crashes to prevent potential data corruption or hardware damage.

What BSOD tells you:

  • Stop code: Error identifier (e.g., 0x0000007B)
  • Error name: Descriptive name (e.g., INACCESSIBLE_BOOT_DEVICE)
  • Failed file: Driver or system file that caused crash
  • QR code: Links to Microsoft support page (Windows 10/11)
üí° Important: Write down or photograph the error code before computer restarts. This is crucial for diagnosis.

Most Common BSOD Error Codes

1. CRITICAL_PROCESS_DIED

Cause: Essential system process terminated unexpectedly

Common triggers: Corrupted system files, driver issues, failing hard drive

2. SYSTEM_SERVICE_EXCEPTION

Cause: System service encountered exception it couldn't handle

Common triggers: Outdated/incompatible drivers, especially graphics and network drivers

3. PAGE_FAULT_IN_NONPAGED_AREA

Cause: System tried to access invalid memory address

Common triggers: Faulty RAM, driver issues, antivirus software conflicts

4. IRQL_NOT_LESS_OR_EQUAL

Cause: Driver accessed memory at wrong priority level

Common triggers: Driver conflicts, faulty drivers, hardware issues

5. DRIVER_IRQL_NOT_LESS_OR_EQUAL

Cause: Driver attempted illegal operation

Common triggers: Network adapter drivers, outdated drivers

8 Solutions to Fix Blue Screen

1 Boot into Safe Mode

Why Safe Mode? Loads Windows with minimal drivers and services, allowing you to troubleshoot without crashes.

How to enter Safe Mode:

  1. If Windows starts: Settings ‚Üí Update & Security ‚Üí Recovery ‚Üí Restart now
  2. Choose Troubleshoot ‚Üí Advanced options ‚Üí Startup Settings ‚Üí Restart
  3. Press F4 for Safe Mode or F5 for Safe Mode with Networking

If Windows won't start:

  1. Force shutdown 3 times (hold power button 10 seconds each time)
  2. Windows will enter Automatic Repair
  3. Follow same steps as above
üí° In Safe Mode: You can uninstall problematic drivers, run system scans, and restore system to earlier point.

2 Update or Rollback Drivers

Most common culprits: Graphics drivers, network adapters, chipset drivers

Update drivers (if BSOD started recently):

  1. Boot into Safe Mode with Networking
  2. Device Manager ‚Üí Expand categories
  3. Right-click device ‚Üí Update driver
  4. Search automatically for drivers
  5. Focus on: Display adapters, Network adapters, Storage controllers

Rollback drivers (if BSOD started after update):

  1. Device Manager ‚Üí Right-click device
  2. Properties ‚Üí Driver tab
  3. Roll Back Driver
  4. Restart computer
‚ö†Ô∏è Pro tip: If error message mentioned specific .sys file (e.g., nvlddmkm.sys), Google it to identify which driver it belongs to.

3 Run Windows Memory Diagnostic

Why test RAM? Faulty RAM is responsible for 25% of BSOD errors, especially PAGE_FAULT errors.

How to run test:

  1. Search "Windows Memory Diagnostic" in Start menu
  2. Restart now and check for problems
  3. Computer will restart and run test (takes 5-15 minutes)
  4. Results shown after restart

If errors found:

  • Test RAM sticks individually (remove all but one, test, swap)
  • Try RAM in different slots
  • Replace faulty RAM modules

Advanced test (more thorough):

Download MemTest86 (free) - runs from USB, more comprehensive than Windows tool.

4 Check Disk for Errors

Why it matters: Corrupted system files or bad sectors cause BSOD, especially CRITICAL_PROCESS_DIED.

Run CHKDSK:

  1. Open Command Prompt as Administrator
  2. Type: chkdsk C: /f /r
  3. Press Y when asked to schedule on restart
  4. Restart computer
  5. Scan runs before Windows loads (takes 30-60 minutes)

Run SFC (System File Checker):

  1. Command Prompt as Administrator
  2. Type: sfc /scannow
  3. Wait for scan to complete (10-30 minutes)
  4. If errors found and fixed, restart

Run DISM (if SFC fails):

  1. Command Prompt as Administrator
  2. Type: DISM /Online /Cleanup-Image /RestoreHealth
  3. Wait 15-45 minutes
  4. After completion, run SFC again

5 System Restore

When to use: BSOD started after installing software, driver, or Windows update.

How to restore:

  1. Boot into Safe Mode or Recovery Environment
  2. Search "Create a restore point"
  3. System Protection ‚Üí System Restore
  4. Choose restore point before BSOD started
  5. Follow wizard, restart after completion
üí° What gets restored: System files, drivers, registry. Your personal files are NOT affected.

No restore points? System Restore may be disabled. You'll need to fix issue manually or reset Windows.

6 Uninstall Recent Updates

If BSOD started after Windows Update:

  1. Settings ‚Üí Update & Security ‚Üí Windows Update
  2. View update history
  3. Uninstall updates
  4. Select recent update ‚Üí Uninstall
  5. Restart computer

Prevent automatic reinstall:

  1. Download "Show or hide updates" troubleshooter from Microsoft
  2. Run tool ‚Üí Hide updates
  3. Select problematic update

7 Check for Overheating

Heat-related BSODs: System crashes to prevent hardware damage from high temperatures.

Quick checks:

  • Feel bottom/sides of computer - extremely hot?
  • Fans running loudly before crash?
  • BSOD happens during gaming/intensive tasks?

Solutions:

  • Clean air vents with compressed air
  • Use cooling pad for laptops
  • Replace thermal paste (if 3+ years old)
  • Ensure proper ventilation around desktop

Monitor temperatures: Download HWMonitor or Core Temp to check CPU/GPU temps. Critical if exceeding 90¬∞C.

8 Reset Windows (Last Resort)

When everything else fails and BSOD persists.

Two options:

  • Keep my files: Removes apps and settings, keeps personal files
  • Remove everything: Complete fresh start (backup first!)

How to reset:

  1. Settings ‚Üí Update & Security ‚Üí Recovery
  2. Reset this PC ‚Üí Get started
  3. Choose option ‚Üí Follow wizard
  4. Process takes 1-3 hours
‚ö†Ô∏è Before resetting: Backup important files to external drive or cloud. List of installed programs (you'll need to reinstall them).

How to Prevent Future BSOD

  • Keep Windows updated: Install updates regularly (but wait 1-2 weeks after major updates for bug fixes)
  • Update drivers monthly: Especially graphics, chipset, network drivers
  • Use quality hardware: Cheap RAM and PSU are common BSOD causes
  • Monitor temperatures: Clean system every 6 months, replace thermal paste every 2-3 years
  • Enable System Restore: Creates recovery points automatically
  • Run maintenance: Disk cleanup, defrag (HDD only), malware scans monthly
  • Stress test new hardware: Test RAM with MemTest86 before normal use
  • Avoid overclocking: Unless you know what you're doing - unstable OC causes crashes

When to Seek Professional Help

Contact technician if:

  • BSOD happens constantly (multiple times per hour)
  • Tried all solutions, still crashing
  • Error points to hardware (RAM test fails, hard drive errors)
  • Computer under warranty (don't open case yourself)
  • Not comfortable with advanced troubleshooting
